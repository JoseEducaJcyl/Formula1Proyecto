# Gestor de Pilotos Fórmula 1

## Descripción
Aplicación de consola en Java que gestiona un registro de pilotos de Fórmula 1. Permite cargar datos desde un archivo, añadir nuevos pilotos, buscar por nombre y guardar los cambios en un nuevo archivo.

## Funcionalidades
- Cargar pilotos desde archivo "pilotos_f1.txt"
- Mostrar todos los pilotos registrados
- Añadir nuevos pilotos (con comprobación de duplicados)
- Buscar pilotos por nombre
- Guardar datos actualizados en "pilotos_f1_nuevo.txt"

## Tecnologías
- Java
- ArrayList
- BufferedReader / FileReader
- BufferedWriter / FileWriter
- Manejo de excepciones IOException
- Scanner
